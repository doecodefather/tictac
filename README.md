# Tic Tac Toe

Ein einfaches Tic Tac Toe Spiel für zwei Spieler auf einem Gerät.  
Dieses Spiel wurde mit **HTML**, **CSS** und **JavaScript** erstellt und ist ideal für Anfänger.

---

## Funktionen

- Zwei-Spieler-Modus (X und O)  
- Gewinner- und Unentschieden-Erkennung  
- Reset-Button zum Neustarten des Spiels  

---

## Nutzung

1. Repository klonen oder ZIP herunterladen  
2. Dateien entpacken (falls ZIP)  
3. `index.html` in einem Browser öffnen  
4. Spielen und Spaß haben!  

---

## Technologie

- **HTML** – Spielfeldstruktur  
- **CSS** – Styling  
- **JavaScript** – Spiel-Logik  

---

## Lizenz
